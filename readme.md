# Juxtapose
## 1. Intro
基于 `SourceGenerator` 的硬编码 `.Net` 多`进程`运行库。

## 2. Requirement
 - .Net5.0+(其它版本没有尝试过)

## 3. 使用方法
```XML
<ItemGroup>
  <PackageReference Include="Juxtapose" Version="1.0.*-*" />
  <PackageReference Include="Juxtapose.SourceGenerator" Version="1.0.*-*" />
</ItemGroup>
```
